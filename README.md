# Расчётник ФОТ РФ — 2026

Автономный веб-калькулятор. Сервер и установка не нужны.

## Запуск

1. Откройте файл `index.html` в Chrome, Edge, Firefox или Safari.
2. Введите оклад, премию и прочие выплаты **чистыми — на руки**.
3. Калькулятор автоматически восстановит начисление до НДФЛ и рассчитает взносы работодателя.
4. Данные автоматически сохраняются в локальном хранилище браузера.
5. Кнопка **Скачать JSON** сохраняет расчёт в файл; **Загрузить JSON** восстанавливает его.

## Как изменить зарплату с июля

1. В поле **Месяц** выберите **Июль**.
2. В основной строке сотрудника введите новый оклад чистыми.
3. Новый оклад автоматически применяется с июля по декабрь.
4. Январь–июнь сохраняют прежний оклад.

Кнопка **⚙** открывает полный график выплат на 12 месяцев. Там можно отдельно изменить любой месяц или скопировать значения выбранного месяца до декабря либо на весь год.

Поведение основных полей:

- **Оклад** — изменение действует с выбранного месяца до декабря.
- **Премия** — изменение действует только в выбранном месяце.
- **Прочее** — изменение действует только в выбранном месяце.

## Что рассчитывается

- начисленная сумма до НДФЛ обратным расчётом от желаемой выплаты на руки;
- прогрессивный НДФЛ 13/15/18/20/22%;
- единый тариф страховых взносов 30% / 15,1%;
- льготные режимы МСП 15% и обрабатывающего производства 7,6% сверх 1,5 МРОТ;
- взносы на травматизм по 32 классам риска;
- дополнительный пенсионный тариф;
- минимальная база в размере МРОТ для руководителя коммерческой организации;
- полная стоимость сотрудника для работодателя за выбранный месяц;
- годовой прогноз по каждому сотруднику и по всей команде на основании помесячного графика;
- последовательный учёт прогрессивного НДФЛ и годового лимита страховых взносов.

## Термины в интерфейсе

- **Оклад / премия / прочее (чистыми)** — сколько сотрудник должен получить на руки.
- **Начислено (грязными)** — сумма до удержания НДФЛ, которую калькулятор подбирает автоматически.
- **К выплате (чистыми)** — сумма на руки, заданная пользователем.
- **Стоимость / месяц** — начисленная сумма плюс все страховые взносы работодателя за выбранный месяц.
- **Стоимость / год** — итог по помесячному графику сотрудника.

## Совместимость JSON

Текущий формат имеет `schemaVersion: 4`. Помесячные значения хранятся в массиве `monthlyCompensation` из 12 элементов — от января до декабря.

JSON версий 1–3 автоматически преобразуются при загрузке. Старые постоянные суммы распространяются на все 12 месяцев, поэтому прежние расчёты не теряются.

## Ввод сумм

Основные денежные поля работают как обычный текстовый ввод с цифровой клавиатурой на телефоне. Стрелки у числовых полей скрыты, а прокрутка колёсиком не меняет значения. Можно вводить `100000`, `100 000` или `100000,50`.

## Ограничения

Калькулятор предназначен для предварительного управленческого расчёта. Для отдельных выплат, иностранцев, нерезидентов, районных коэффициентов и специальных льгот правила могут отличаться. Проверьте результат у бухгалтера.
